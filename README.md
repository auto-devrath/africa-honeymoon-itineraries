# October Africa Honeymoon

Interactive React/Vite site for comparing three October honeymoon itinerary concepts from the Gemini Deep Research report:

- Egypt + Kenya
- Namibia + Victoria Falls
- Egypt + Victoria Falls

## Local Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The production build is emitted to `dist/` and uses relative asset paths so it can be served from GitHub Pages or another static host.
