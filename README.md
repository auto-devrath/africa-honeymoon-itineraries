# October Africa Honeymoon

Interactive React/Vite decision workspace for comparing an October Africa honeymoon longlist and three buildable itinerary concepts from the research reports:

- Egypt + Kenya
- Namibia + Victoria Falls
- Egypt + Victoria Falls

## Local Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The production build is emitted to `dist/` and uses relative asset paths so it can be served from GitHub Pages or another static host.
